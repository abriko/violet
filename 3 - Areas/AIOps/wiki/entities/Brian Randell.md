# Brian Randell

**Role:** Principal Consultant, MCW Technologies; former Microsoft MVP (17+ years); former GitHub employee  
**Affiliation:** Microsoft ecosystem consultant  
**Known for:** Co-author of *Essential DevOps*; keynote speaker on Agentic DevOps

---

## Contributions to This Wiki

Co-presented the Live! 360 keynote "Agentic DevOps in Real Life" (2025-12-22) with [[3 - Areas/AIOps/wiki/entities/Mickey Gousset]]. Key contributions:

- Defined **Agentic DevOps** as collaborative, not merely automated
- Articulated the **Human-in/on/out-of-the-Loop** taxonomy for AI-human collaboration
- Presented the **productivity paradox**: AI feels faster but may not ship more value without discipline
- Proposed the **30/60/90-day rollout plan** for enterprise Agentic DevOps adoption
- Demoed Azure SUR Agent (autonomous cloud monitoring preview)
