# LangChain

**Type:** Open-source framework / company
**Founded by:** [[3 - Areas/AIOps/wiki/entities/Harrison Chase]]
**Website:** https://langchain.com

---

## What It Is

LangChain is an open-source framework for building LLM-powered applications. The company also develops two complementary products for agent systems:

### LangGraph

Agent orchestration framework with durable execution. Key design principle: **no hidden prompts** — developers retain full control over [[3 - Areas/AIOps/wiki/concepts/Context Engineering]] with no framework-injected prompt content. Supports long-running agent tasks with fault tolerance and checkpointing.

### LangSmith

Agent debugging, observability, and [[3 - Areas/AIOps/wiki/concepts/LLM Evaluation]] platform. Provides tracing, monitoring, and eval tooling for [[3 - Areas/AIOps/wiki/concepts/Multi-Agent Architecture]] systems.
