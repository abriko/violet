**Role:** Senior Director, Developer Tools, Microsoft

## Contributions to This Wiki

Co-presented the Microsoft Reactor session "The future of software creation with Agentic DevOps" (2026-01-28) with Patrick (GitHub). Key contributions:

- Defined the **Three Pillars of Agentic DevOps** framework (developer experience, agents across SDLC, AI building AI)
- Articulated the **GitHub Copilot evolution timeline** from 2021 to 2026
- Framed the developer role shift: pair programmer → peer programmer → AI teammate → manager/orchestrator
- Cited Microsoft internal metrics: 91% engineering adoption, SRE agent saving 10,500 hours + 7K incidents
- Described the VS Code team as a live case study: monthly → daily releases enabled by Copilot
