# Harness

**Type:** DevOps Platform / Vendor  
**Website:** https://www.harness.io

---

## Overview

Harness is a software delivery platform focused on CI/CD, feature flags, cloud cost management, and increasingly, Agentic DevOps capabilities. In the context of this wiki, Harness is notable as the author of an architect's guide that introduced the [[3 - Areas/AIOps/wiki/concepts/Three Horizons of Adoption]] and [[3 - Areas/AIOps/wiki/concepts/Policy as Code]] guardrails frameworks for autonomous infrastructure.

---

## Key Contributions to This Wiki

- **C-P-A Model** framing (Perception, Memory, Reasoning, Tool Use) — formal anatomy of a DevOps agent
- **Three Horizons of Adoption**: Augmented Operator → Agent Swarms → Autonomous SRE
- **Guardrails Architecture**: Policy-as-Code + Contextual Permissions + Black Box Recorder
- **Harness AI (AIDA)**: native AI assistant embedded in the Harness platform for DevOps tasks

---

## Positioning

Harness positions agentic AI as a transition from "Automated" (deterministic, brittle) to "Autonomous" (probabilistic, adaptive). The core thesis: engineers evolve from YAML authors to Cognitive Architecture designers.
