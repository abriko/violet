# CNCF

**Full name:** Cloud Native Computing Foundation  
**Type:** Open-source software foundation  
**Website:** https://www.cncf.io  
**Parent:** Linux Foundation

---

## Overview

The Cloud Native Computing Foundation (CNCF) is a Linux Foundation project that hosts and nurtures open-source cloud-native computing projects. It manages projects through a maturity lifecycle: **Sandbox → Incubating → Graduated**.

CNCF hosts KubeCon + CloudNativeCon, one of the largest open-source conferences, which is a primary venue for AI-with-observability discourse.
