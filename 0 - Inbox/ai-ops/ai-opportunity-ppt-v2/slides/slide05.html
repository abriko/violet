Execution principle: lead with UC1 now; let UC2 and UC3 build on the same agent and service context foundation.

Primary track · phase 1 focus

UC1 · Delivery & Release Intelligence

Most direct path to user-visible value.

  • Pipeline AI L1 is already live.
  • L2 / L3 and miniops 2.0 are the most practical next steps.
  • This is the wedge that proves platform leverage in daily engineering work.

Shared layer next

UC2 · Monitoring to Recovery

Observe, diagnose, then automate safely.

  • Proactive monitoring signal intake.
  • RCA assistance and evidence chain.
  • Non-production auto-fix as the safe proving ground.

Shared layer next

UC3 · Cost Intelligence

Operational efficiency with financial visibility.

  • Cloud cost observability and tagging discipline.

  • Daily agent pattern once the data quality foundation is ready.
