// 5 Design Principles

01

Security as First-Class Signal

CVE / IaC findings enter every RCA — not patched in after the fact

02

Reasoning ≠ Execution

LLM reasons only. All writes require Policy Gate + human approval

03

Evidence-Linked RCA

Every conclusion cites a specific log line or deployment event — no black boxes

04

Progressive Autonomy

AI cannot self-escalate. Non-prod first. Reversible actions first.

05

Agent Observability

Every session logs: prompt, tool calls, model version, token cost
